# Muhasebe Programı Pro

Modern, Python tabanlı masaüstü Ön Muhasebe ve Stok Takip Programı.

## Özellikler

### Mevcut Modüller

- Kullanıcı Girişi
- Dashboard
- Stok Yönetimi
- Cari Yönetimi
- Satış Yönetimi
- Satış Geçmişi
- Alış Yönetimi
- Kasa Yönetimi

## Geliştirilecek Modüller

- Dashboard Pro
- Raporlar
- Grafikler
- PDF Fatura
- Excel İçe / Dışa Aktarma
- Barkod Sistemi
- Firma Ayarları
- Kullanıcı Yetkilendirme
- Otomatik Yedekleme

## Kullanılan Teknolojiler

- Python 3
- CustomTkinter
- SQLite
- ttk Treeview

## Kurulum

```bash
git clone https://github.com/semihekini1/MuhasebeProgrami.git

cd MuhasebeProgrami

pip install -r requirements.txt

python main.py
```
